# QuantumCoders

QuantumCoders is a team project created for the **NASA Space Apps Challenge 2025**, an international hackathon focused on solving real-world challenges using open data and technology.

## About the Project
This repository contains the work developed during the NASA Space Apps Challenge, where the goal was to design and prototype a solution within a limited time frame while collaborating as a team.

The project focuses on problem-solving, data exploration, and building a functional concept aligned with the challenge requirements.

## Technologies
- Python
- Web technologies
- APIs and open data sources

## Team
QuantumCoders is a collaborative team formed specifically for the NASA Space Apps Challenge.

## Project Status
This project represents a hackathon prototype and may be further improved or refined after the event.

## Purpose
The goal of this project was to gain experience in collaborative development, rapid prototyping, and working on a real-world challenge in a competitive environment.
